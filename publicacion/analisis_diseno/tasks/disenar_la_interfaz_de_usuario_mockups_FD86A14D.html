Task: Disenar la interfaz de usuario (mockups)
Por ultimo se realiza un prototipado de lo desado por el cliente, segun las especificaciones anteriormente suministradas (mockups)
Purpose

El objetivo de diseñar la interfaz de usuario es crear una experiencia visual y funcionalmente atractiva que permita a los usuarios interactuar de manera intuitiva con un sistema o aplicación.

Relationships
RolesPrimary Performer:

Outputs
  • Interfaces de usuario o mockups
    Main Description
     La interfaz de usuario es la capa visible con la que los usuarios interactúan al utilizar un software, una página web o una aplicación. El diseño de la interfaz busca crear una estructura visual y funcional que organice y presente la información de manera clara
    Key Considerations
    Comprender a los usuarios, Definir la paleta de colores y el estilo visual
