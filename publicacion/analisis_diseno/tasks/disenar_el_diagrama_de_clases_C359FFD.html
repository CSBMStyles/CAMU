Task: Disenar el diagrama de clases
Por ultimo se realiza el diagrama de clases
Purpose
El objetivo de diseñar un diagrama de clases es visualizar y representar la estructura estática de un sistema de software, mostrando las clases y las relaciones entre ellas.
Relationships
RolesPrimary Performer:
  • Arquitecto de software
Additional Performers:
Outputs
  • Diagrama de clases
    Main Description
    El diagrama de clases es una herramienta de modelado visual utilizada en la ingeniería de software para representar las clases, los atributos y los métodos de un sistema
    Key Considerations
    Definir atributos y métodos, Identificar las relaciones, Representar las clases y relaciones
