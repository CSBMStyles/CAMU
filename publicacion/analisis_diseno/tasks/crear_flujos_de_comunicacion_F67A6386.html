Task: Crear flujos de comunicacion
Una vez identicadas las clases se crean los flujos de comunicación que hay entre ellas de manera cohrente.
Purpose
 El objetivo de crear un flujo de comunicación es modelar y visualizar cómo la información y los mensajes se transmiten entre los componentes o sistemas de un sistema de software
Relationships
RolesPrimary Performer:
  • Arquitecto de software
Additional Performers:
Outputs
  • Diagrama de secuencias
    Main Description
    Un flujo de comunicación representa la secuencia y la dirección de los mensajes y la información entre los componentes o sistemas del sistema de software. Se utiliza para mostrar cómo se intercambian los datos, las solicitudes, las respuestas y otros tipos de mensajes en el sistema.
    Key Considerations
    Flujo de comunicación, información, mensajes, componentes, sistemas, interacción, diagrama de secuencia, diagrama de actividad, secuencia, dirección.
