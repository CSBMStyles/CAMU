Task: Crear el diagrama de comunicacion
Por ultimo, con la información anteriormente recopilada se crea el diagrama de comunicación
Purpose
El objetivo de crear un diagrama de comunicación es representar de forma visual las interacciones y los mensajes que ocurren entre los objetos o componentes del sistema
Relationships
RolesPrimary Performer:
  • Arquitecto de software
Additional Performers:
Outputs
  • Diagrama de secuencias
    Main Description

    Este diagrama permite comprender cómo se comunican entre sí los distintos elementos del sistema, mostrando el flujo de información y los intercambios de mensajes que ocurren durante la ejecución.

    El diagrama de comunicación se basa en la perspectiva de los objetos y las interacciones entre ellos.

    Key Considerations
     Diagrama de comunicación, interacciones, mensajes, objetos, componentes, flujo de información, secuencia, colaboración.
