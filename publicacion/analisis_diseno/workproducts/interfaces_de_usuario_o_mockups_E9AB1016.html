Artifact: Interfaces de usuario o mockups
Es el prototipado visual mas cercano al producto final
Purpose
El objetivo de las interfaces de usuario o mockups es visualizar y representar de manera gráfica cómo se verá y funcionará la aplicación final antes de su desarrollo completo. Proporciona una vista previa de la interfaz de usuario, la disposición de los elementos y la navegación, permitiendo evaluar y validar el diseño antes de la implementación.

Description
Main Description Las interfaces de usuario o mockups son representaciones estáticas o interactivas de la interfaz gráfica de una aplicación. Estas representaciones suelen ser creadas utilizando herramientas de diseño gráfico o prototipado, y se centran en mostrar cómo se estructuran los elementos visuales y cómo interactúan entre sí
