Artifact: Diagrama de secuencias
Es una representación visual que muestra la interacción y el orden temporal de objetos o componentes del sistema.
Purpose
El objetivo de un diagrama de secuencia es visualizar y representar la interacción y el flujo de mensajes entre los objetos en un escenario específico dentro de un sistema. Este tipo de diagrama permite comprender la secuencia de eventos y la colaboración entre los objetos, lo que ayuda a diseñar y comprender el comportamiento dinámico de un sistema de software.

Description
Main Description Un diagrama de secuencia es una representación gráfica que muestra la secuencia de eventos y las interacciones entre objetos en un escenario particular dentro de un sistema. Se utiliza para modelar el flujo de mensajes entre los objetos y capturar el comportamiento dinámico de un sistema.
