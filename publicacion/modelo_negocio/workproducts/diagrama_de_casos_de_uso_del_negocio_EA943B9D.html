Artifact: Diagrama de casos de uso del negocio
Es una representación visual que muestra las interacciones entre los actores externos y los casos de uso principales en el contexto de un negocio.
Purpose
Representación visual que describe las interacciones entre los actores (usuarios o sistemas externos) y el sistema en el contexto de un determinado negocio

  • Recopilar requisitos del negocio
Description
Main Description Este diagrama es una herramienta fundamental en el análisis de requerimientos y en el diseño de sistemas, ya que permite identificar y comprender los diferentes casos de uso que deben ser soportados por el sistema.
