Task: Crear rasgos o perfil de usuario
Crear rasgos o perfil de usuario es un proceso para definir las características y necesidades de los usuarios finales del sistema.
Purpose
Crear un perfil detallado de los usuarios finales del sistema, para poder diseñar una solución que satisfaga sus necesidades de manera efectiva
Relationships
RolesPrimary Performer:
  • Analista de negocio
Additional Performers:
InputsMandatory:
  • Perfil de stakeholders
    		Optional:
    • None
    Outputs
    • Lista actores de sistema
      Main Description
      Se busca identificar las características demográficas, psicológicas y de comportamiento de los usuarios, así como sus necesidades y expectativas en relación al sistema, siendo un soporte para los requierimientos
