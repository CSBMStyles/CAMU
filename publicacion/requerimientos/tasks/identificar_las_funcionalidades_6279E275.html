Task: Identificar las funcionalidades
Se identifican las funcionalidades que seran implementadas en el proyecto
Purpose
Identificar de manera exhaustiva todas las funcionalidades requeridas por el sistema, es decir, las acciones o tareas que el sistema debe ser capaz de realizar
Relationships
RolesPrimary Performer:
  • Arquitecto de software
Additional Performers:
InputsMandatory:
  • Documento de requisitos funcionales
    		Optional:
    • None
    Outputs
    • Analisis de funcionalidades
      Main Description
      Identificar las funcionalidades es un paso esencial en el proceso de desarrollo de software. donde se analiza y definir las diferentes características y capacidades que debe tener el sistema para cumplir con los requisitos.
