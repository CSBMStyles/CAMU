Task: Priorizar las funcionalidades
Se priorizan las funcionalidades por tiempo de entrega, como cuales se deben realizar rapidamente y en que plazo
Purpose
Tomar decisiones informadas sobre qué funcionalidades se deben implementar primero y cuáles pueden ser pospuestas o eliminadas
Relationships
RolesPrimary Performer:
  • Arquitecto de software
Additional Performers:
InputsMandatory:
  • Documento de requisitos funcionales
    		Optional:
    • None
    Outputs
    • Analisis de funcionalidades
      Main Description
      Establecer un orden de importancia o prioridad para las diferentes funcionalidades identificadas en el sistema. Durante esta actividad, el equipo de desarrollo evalúa y clasifica las funcionalidades en función de su valor, impacto y necesidad, con el objetivo de asignar recursos y tiempo de manera efectiva.
