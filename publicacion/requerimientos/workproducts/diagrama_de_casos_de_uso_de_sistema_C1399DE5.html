Artifact: Diagrama de casos de uso de sistema
Es una representación gráfica que muestra las interacciones entre los actores implicados y el sistema.
Purpose
Representar de manera clara y concisa las funcionalidades y acciones que un sistema de software debe realizar, y cómo los usuarios o sistemas externos interactúan con él.

Description
Main Description

El diagrama de casos de uso de sistema consta de varios elementos, incluyendo actores, casos de uso y relaciones.

Los actores representan a los usuarios o sistemas externos que interactúan con el sistema, mientras que los casos de uso representan las funcionalidades y acciones que el sistema debe realizar
