Artifact: Formulario de necesidades
formulario donde se toman las peticiones del cliente
Purpose
Proporcionar una estructura clara y sistemática para capturar la información esencial que guiará el proceso de diseño, desarrollo y entrega de una solución
Relationships
RolesResponsible:
  • Analista de negocio
Modified By:
  • Analista de negocio
TasksInput To:
  • Crear matriz de trazabilidad
  • Identificar los requisitos del proyecto
  • Ordenar requisitos por prioridad
Output From:
  • Hablar con cliente sobre las necesidades que tiene
Description
Main Description Documento estructurado diseñado para recopilar y registrar las necesidades y requerimientos de los usuarios o stakeholders en relación a un determinado proyecto, producto o servicio, se utiliza como una herramienta para recopilar información relevante y precisa sobre las necesidades específicas que deben abordarse en el desarrollo de una solución.
