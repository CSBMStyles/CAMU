Artifact: Matriz de trazabilidad
Es una herramienta que sirve en la gestión de requisitos para rastrear y documentar la relación entre los diferentes elementos del proyecto, en este caso los requisitos.
Purpose
Proporcionar una visión clara y estructurada de cómo cada elemento está relacionado entre sí y cómo contribuye al logro de los objetivos del proyecto.

Description
Main Description Una matriz de trazabilidad es una herramienta utilizada en la gestión de proyectos para establecer y rastrear las relaciones entre diferentes elementos del proyecto, como requisitos, casos de uso, pruebas, componentes y entregables
