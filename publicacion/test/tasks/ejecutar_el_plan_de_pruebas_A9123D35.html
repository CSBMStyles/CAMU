Task: Ejecutar el plan de pruebas
Una vez se tenga el entorno de pruebas se ejecuta el plande pruebas para identificar errores.
Purpose
El objetivo de esta tarea es ejecutar un plan de pruebas predefinido para asegurar la calidad y el correcto funcionamiento del software.
Relationships
RolesPrimary Performer:
  • Desarrollador Backend
  • Desarrollador Frontend
Additional Performers:
InputsMandatory:
  • Codigo del proyecto backend
  • Codigo del proyecto frontend
    		Optional:
    • None
    Outputs
    • Informe de errores
      Main Description
      El desarrollador backend y el desarrollador frontend colaboran para ejecutar las pruebas planificadas, incluyendo pruebas unitarias, pruebas de integración y pruebas de aceptación. Siguen el plan de pruebas establecido, registran los resultados y documentan cualquier defecto o error encontrado durante las pruebas.
      Key Considerations
       Plan de pruebas, ejecución de pruebas, pruebas unitarias, pruebas de integración, pruebas de aceptación, documentación de defectos.
