Task: Realizar pruebas de despliegue
Se realizan pruebas del despliegue del software
Purpose
El objetivo de esta tarea es verificar y asegurar que la aplicación se despliegue correctamente en el entorno de Azure, y que todas las funcionalidades estén operativas.

      Main Description
      Durante las pruebas de despliegue, tanto el desarrollador backend como el desarrollador frontend se aseguran de que la aplicación esté funcionando correctamente en el entorno de Azure. Realizan pruebas exhaustivas para verificar que todas las funcionalidades y características estén operativas después del despliegue
