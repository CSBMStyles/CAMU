Task: Depuracion y seguimiento de errores
Como paso adicional se contempla posibles errores a futuro, asi que hace un seguimiento del software desplegado.
Purpose
 El objetivo de esta tarea es identificar y solucionar errores o fallos en la aplicación, asegurando su correcto funcionamiento y proporcionando una experiencia libre de errores a los usuarios.
Relationships
RolesPrimary Performer:
  • Desarrollador Backend
  • Desarrollador Frontend
Additional Performers:
InputsMandatory:
  • Aplicacion final
    		Optional:
    • None
    Outputs
    • Informe de despligue
      Main Description
      Tanto el desarrollador backend como el desarrollador frontend se dedican a la depuración y seguimiento de errores. Realizan pruebas exhaustivas, analizan los problemas encontrados, identifican la causa raíz y aplican soluciones para resolver los errores.
      Key Considerations
       Desarrollador backend, desarrollador frontend, depuración, seguimiento de errores, pruebas, soluciones.
