Artifact: Codigo del proyecto backend
Codigo fuente que contiene la logica del negocio.
Purpose

traducir los requisitos funcionales del proyecto en código ejecutable que implemente la lógica de negocio del sistema. Los desarrolladores backend se encargan de construir los algoritmos, las reglas y los procesos necesarios para que el software realice las operaciones requeridas

Description
Main Description  Esta tarea implica la implementación del código fuente del proyecto de software en el backend. Los desarrolladores backend utilizan lenguajes de programación como Python, Java, C#, entre otros, para construir la lógica de negocio, manipular y almacenar datos, y establecer la comunicación con otros componentes del sistema
