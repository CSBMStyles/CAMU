Artifact: Informe de validaciones
Documento que contiene las fases evaluativas por las que pasa el desarrollo backend
Purpose
 El objetivo principal del informe de validaciones es evaluar si el software cumple con los requisitos funcionales y no funcionales establecidos durante la fase de análisis y diseño. Se verifica que todas las funcionalidades estén implementadas correctamente y que se comporten según lo esperado.

Description
Main Description El informe de validaciones en un proyecto de software tiene como objetivo documentar y presentar los resultados de las pruebas y validaciones realizadas en el software. Este informe es elaborado por el equipo de pruebas y validación, y su propósito es evaluar si el software cumple con los requisitos y está listo para su implementación o entrega.
